查看版本日志信息

  • 查看某个分支仓库中的版本信息
  • 感觉日志信息输出太多，可以使用 git log –pretty=oneline 在一行显示日志

回滚到指定版本

  • 使用 git reset –hard 指令进行版本回滚
  • 使用 git reflog 指令查看刚才所有操作日志

git log 与git reflog 区别

  • git log：只查询文件修改/新增/删除等与文件有关的日志记录；
  • git reflog：不仅记录与文件有关的记录，还会记录对分支仓库的回滚记录。
